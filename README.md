What is the Library Management System?
The Library Management System (LMS) is a complete web-based prototype designed to replace manual library operations with a modern digital solution. It is built using HTML, CSS, and JavaScript and serves as a fully functional front-end prototype that demonstrates how a real library system would work.

The system handles all core library functions including book management, member registration, loan processing, fine calculation, and reporting — all through an intuitive, user-friendly interface.
Purpose and Objectives

Why Was This System Built?
The system was developed to address the common challenges faced by libraries using paper-based or outdated methods:
- Manual book tracking → Books get lost or misplaced
- Paper records → Slow and error-prone
- No automated fines → Missed revenue and unfair to members
- No search functionality → Members struggle to find books
- No reporting → No insight into library activity 

 Who Uses the System?
 
1. Librarian (Administrator)
Full access to all system functions
Can add, edit, and delete books
Can register, update, and delete members
Can issue and process book returns
Can manage fines and generate reports

2. Member (Library User)
Limited self-service access
Can search the book catalogue
Can view currently borrowed books
Can view borrowing history
Can check outstanding fines
